PHPFCGI

What is phpfcgi?

Basically, it's a simple FastCGI server written in pure PHP that tries to duplicate a more standard FastCGI runloop ala Python, Perl, or Ruby.

Example

Simple hello world FCGI server

include 'FCGI.php';
$server = new FCGI_Server();
while($req = $server->Accept())
{
    echo "Hello World";
}
